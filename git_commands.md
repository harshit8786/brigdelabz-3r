GIT COMMANDS
git clone <repo_url> - GitHub repository ko local system me copy karta hai

git status - Working directory aur staging area ka status dikhata hai

git add . - Saare modified files ko staging area me bhejta hai

git add file.txt - Sirf ek specific file ko staging me add karta hai

git commit -m "message" - Staged files ko save karta hai commit ke roop me

git log - Commit history dikhata hai

git log --oneline - Short commit history dikhata hai

git branch - Saari branches list karta hai

git checkout branch_name - Branch switch karta hai

git checkout -b branch_name - New branch bana kar usme switch karta hai

git switch branch_name - Branch switch karne ka modern command

git pull origin main - Remote se latest code laata hai

git push origin branch_name - Local commits ko GitHub par bhejta hai

git push -u origin branch_name - First time branch ko GitHub par push karta hai

git merge branch_name - Dusri branch ko current branch me merge karta hai

git branch -d branch_name - Branch delete karta hai

git reset --hard HEAD - Last commit tak project reset karta hai

git remote -v - Remote repository ka URL dikhata hai
